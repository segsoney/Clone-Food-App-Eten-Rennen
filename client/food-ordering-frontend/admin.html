What you can do here

  • Menu management — Add new dishes, edit prices and details, or remove items. Slugs must match storefront cart IDs (e.g. crash-burger).
  • Order oversight — See every customer order: items, totals, payment method and payment status.
  • Fulfillment workflow — Move orders through Received → Preparing → Out for delivery → Delivered, or mark Cancelled.
  • Access control — Only accounts with the admin role (from npm run seed:admin) can use this dashboard; the API checks your JWT.
