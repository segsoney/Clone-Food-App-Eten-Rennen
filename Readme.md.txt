# ETENRENNEN - Food Ordering Website

ETENRENNEN is a full-stack food ordering web application developed as an internship project.

## Features

- User Authentication
- Food Menu
- Cart System
- Order Placement
- Responsive Design
- Backend API Integration

## Tech Stack

Frontend:
- React
- Tailwind CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Deployment

Frontend: Vercel
Backend: Render
Database: MongoDB Atlas
