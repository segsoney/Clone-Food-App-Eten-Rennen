# 🍔 EtenRennen — Food Ordering Website

EtenRennen is a modern, responsive food ordering website inspired by popular delivery platforms like Swiggy and Zomato, combined with a playful gaming theme influenced by Crash Bandicoot 🎮.

It provides a smooth and interactive user experience where users can browse food items, filter by categories, add items to cart, and simulate ordering — all within a clean and responsive UI.

---

## 🚀 Features

* 📱 Fully Responsive Design (Mobile + Desktop)
* 🍔 Category-Based Food Filtering
* 🔍 Search Functionality
* 🛒 Dynamic Cart System (using localStorage)
* 👤 Login/Signup Simulation
* 🎮 Crash Bandicoot Inspired UI Theme
* ✨ Animated Navbar & Interactive UI Elements
* 🧠 Beginner-Friendly Code Structure

---

## 🎯 How It Works

* Users can browse food items on the menu page
* Filter items based on categories
* Add items to the cart
* Cart updates dynamically using localStorage
* Login/signup is simulated for frontend experience
